# AGENTS.md

Локальные правила для новостей в `apps/www/src/data/news`.

## Тексты новостей

- Для КП Шелково использовать краткое наименование: `КП Шелково`.
- В `summary` допустимы более короткие и нейтральные формулировки (например, `В поселке`), если смысл новости сохраняется.
- Не добавлять служебные приветствия и финальные вежливые формулы в body новости (например, `Уважаемые жители!`, `Благодарим за понимание`).
- `seo`, `seo.title` и `seo.description` опциональны: добавлять их только когда обычные `title` или `summary` слишком длинные для поискового сниппета.
- В `seo.title` хранить только уникальную часть заголовка новости, без `— Новости — Шелково Онлайн`: секцию и бренд добавляет шаблон страницы.
- `seo.description` используется только для meta description; не сокращать ради SEO обычный `summary`, если он нужен в карточках, RSS или discovery-данных.
- Для slug/имени markdown-файла новости предпочитать короткий вариант без лишних слов (например, `tick-treatment` вместо `tick-treatment-eco-club`).
- Если точная дата публикации в источнике не дана, в `date` ставить сегодняшнюю дату; отчетный период отражать в заголовке и body, а не подменять им дату публикации.

## Регулярные отчеты ОК Комфорт

- Для регулярных отчетов ОК Комфорт о работах во всех поселках всегда добавлять тег `отчет`.
- В body оставлять общий контекст отчета и блоки, относящиеся к КП Шелково; блоки, посвященные другим поселкам или группам поселков, удалять.
- `Шелково Эко Клаб` в таких отчетах нормализовать как `КП Шелково`; отдельные части поселка допустимо сокращать до `Форест`, `Вилладж`, `Ривер`, если из контекста ясно, что речь о КП Шелково.
- Длинные перечни работ при необходимости структурировать, чтобы новость не превращалась в плоскую простыню пунктов; не добавлять лишнюю вложенную группировку, если короткий перечень читается нормально.
- Если пункт отчета точно связан с уже опубликованной новостью, делать ссылку с подходящей фразы на эту новость; не ссылать общую формулировку на частную новость, если совпадение только тематическое.

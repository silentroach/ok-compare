# AGENTS.md

Локальные правила для новостей в `apps/www/src/data/news`.

## Тексты новостей

- Для КП Шелково использовать краткое наименование: `КП Шелково`.
- В `summary` допустимы более короткие и нейтральные формулировки (например, `В поселке`), если смысл новости сохраняется.
- Не добавлять служебные приветствия и финальные вежливые формулы в body новости (например, `Уважаемые жители!`, `Благодарим за понимание`).
- `seo`, `seo.title` и `seo.description` опциональны: добавлять их только когда обычные `title` или `summary` слишком длинные для поискового сниппета.
- В `seo.title` хранить только уникальную часть заголовка новости, без `— Новости — Шелково Онлайн`: секцию и бренд добавляет шаблон страницы.
- `seo.description` используется только для meta description; не сокращать ради SEO обычный `summary`, если он нужен в карточках, RSS или discovery-данных.
- Для slug/имени markdown-файла новости предпочитать короткий вариант без лишних слов (например, `tick-treatment` вместо `tick-treatment-eco-club`).

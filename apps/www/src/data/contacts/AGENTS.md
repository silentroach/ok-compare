# AGENTS.md

Локальные правила для данных раздела `Полезные контакты`.

- Один Markdown-файл в этой папке равен одному опубликованному контакту.
- Не добавлять реальные контакты без источника от пользователя.
- Frontmatter должен проходить `RawContactSchema`: `title`, `slug`, `category`, `updated_at`, `contacts` обязательны.
- `slug` писать латиницей в lower-case, с цифрами и дефисами.
- `updated_at` хранить как календарную дату `YYYY-MM-DD`.
- В списке контактов не обещать качество услуги: не использовать формулировки `проверенный`, `надежный`, `лучший`, `рекомендуем` от лица сайта.

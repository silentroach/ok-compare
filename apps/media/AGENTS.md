# AGENTS.md

Локальные инструкции для статического приложения `media.kpshelkovo.online`.

- Приложение собирает только локальную страницу `404.html`; публичные медиафайлы nginx проксирует напрямую в S3.
- Не добавлять prefetch, client router, аналитику и клиентские скрипты без отдельного обоснования и изменения CSP.
- Общий UI и брендовые assets брать из `@shelkovo/ui`, не копировать из `apps/www`.
- Локальный dev-server работает на `http://localhost:4322` и обязан завершаться ошибкой, если порт занят.
- Production artifact: `dist/media` в корне workspace.

# Аудит Сайтов Коттеджных Поселков

Дата старта: 2026-05-23.

Этот каталог превращает широкую выборку из `docs/research/cottage-settlement-sites.md` в рабочий аудит. Production-код сайта во время исследования не меняется.

## Что Уже Есть

- `../cottage-settlement-sites.md` — широкая стартовая выборка сайтов и каталогов.
- `../cottage-settlement-agent-brief.md` — бриф, вопросы исследования и структура заметки по сайту.
- `starter-sample.md` — первая сбалансированная выборка из 30 сайтов для ручного аудита.
- `site-note-template.md` — копируемый шаблон одной заметки по сайту.
- `sites/` — место для отдельных заметок по каждому проверенному сайту.

## Как Работать Дальше

- Брать сайты из `starter-sample.md` сверху вниз внутри каждого сегмента.
- Для каждой проверки копировать `site-note-template.md` в `sites/[slug].md` и заполнять по фактам.
- Если сайт мертвый, припаркованный, недоступный или явно неофициальный, фиксировать это в отдельной заметке и брать замену из `../cottage-settlement-sites.md`.
- На каждый сайт создавать отдельный markdown-файл в `sites/` по схеме из `../cottage-settlement-agent-brief.md`.
- Не использовать markdown-таблицы.
- После набора заметок создать `research-report.md`, затем отдельным проходом `synthesis.md`.

## Граница Исследования

- Исследуем структуру сайта, публикации, доверие, пользу для жителей и визуально-контентные паттерны.
- Не ранжируем поселки по качеству жизни или недвижимости.
- Не копируем тексты, визуальные ассеты, макеты и бренд-язык чужих сайтов.
- Не считаем брокерские страницы официальными источниками без подтверждения на самом сайте.

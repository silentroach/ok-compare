PDF: `full`.

## Правила переноса

- Пустая ячейка в PDF перенесена как `-`. Это не означает подтвержденный ноль.
- `Итог` считается только как сумма явно заполненных числовых ячеек. Если сумма может исказить смысл показателя, указано `не суммируется`.
- Названия объектов сохранены близко к цитате из таблицы PDF.
- Строки с неясной единицей измерения, разрывом таблицы между страницами или OCR-неоднозначностью помечены как `требует визуальной сверки`.

## Что требует отдельной проверки

| Тема                        | Почему проверить                                                                                                                         |
| --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| Пустые ячейки               | В таблице много пустых значений. Для публичного слоя нельзя автоматически трактовать их как отсутствие объекта без первичных документов. |
| `ИДН`                       | Единица измерения в строке не заполнена.                                                                                                 |
| Светильник `ЖКУ 16-100-001` | В колонке `Ед. изм.` стоит `100`, а отдельной строкой указана `Мощность светильника`; единица мощности не указана.                       |
| Опоры освещения             | В PDF указано `м²`, хотя для опор ожидаемо количество или погонные метры; переносить только с пометкой.                                  |
| `КТППНккк`                  | Наименование выглядит как OCR-неоднозначность и требует сверки с исходным изображением или первичным документом.                         |
| Охрана, обходы/объезды      | Строка разорвана между pages 129-130; показатель `кол-во раз в сутки` является периодичностью, а не инвентарной суммой.                  |

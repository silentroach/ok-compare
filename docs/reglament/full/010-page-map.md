## Расчетные допущения для отдельной проверки

Эти пункты не разобраны подробно в рамках задачи 010, но страницы стоит передать в задачу `060`:

| Страницы | Фрагмент                                                 | Почему может потребовать проверки                                                                                                                |
| -------- | -------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| 3-5      | Правовая и организационная основа расчета единого тарифа | Регламент утверждает единый расчет для четырех поселков без раздельного учета.                                                                   |
| 5        | Порядок распределения платы за услуги                    | Плата распределяется пропорционально площади земельного участка собственника.                                                                    |
| 5-6      | Формула платы за 1 участок в месяц                       | В формуле указан делитель `Sобщ. уч × 12`; нужно сверять единицы измерения при переходе к соткам и м².                                           |
| 10       | Норматив общеэксплуатационных расходов                   | В документе выбран уровень 70% от расходов на оплату труда основных рабочих и машинистов.                                                        |
| 10       | Плановая прибыль                                         | Прибыль считается от расходов на оплату труда основных работников и машинистов; для публичного слоя формулировать только как `требует проверки`. |
| 123-126  | Общая калькуляция и общая смета                          | Сводные суммы и итоговая плата за 1 сотку могут быть базой для сверки с существующим калькулятором.                                              |

## Заметки по качеству извлечения

| Страницы | Заметка                                                                                                                                                                                   |
| -------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 128-130  | Таблицы Приложения №2 распознаны текстом, но часть пустых ячеек и единиц измерения выглядит неоднозначно. В задаче `020` все такие строки нужно помечать как `требует визуальной сверки`. |
| 131-134  | Схемы поселков являются картами. Их не нужно подробно OCR-ить в текущей цепочке задач.                                                                                                    |
| 136-138  | В нормативной литературе заметны OCR-искажения годов и номеров документов. Этот раздел сейчас пропускается.                                                                               |

# 020: Извлечь состав общего имущества

## Цель

Достать из полного регламента состав общего имущества по поселкам: дороги, ливневки, зеленые насаждения, площадки, электрика, охрана и другие объекты.

Это самый ценный слой для LLM и вероятный кандидат на отдельную публичную страницу.

## Вход

- Контекст: `docs/reglament/full/000-agent-context.md`.
- Карта страниц: `docs/reglament/full/010-page-map.md`.
- Источник: `docs/reglament/original/full.pdf`, Приложение №2.

## Выход

Создать файл:

- `docs/reglament/full/020-common-assets.md`.

## Что извлечь

- Дороги и тротуары.
- Ливневые траншеи.
- Зеленые насаждения и лес.
- Объекты благоустройства.
- Электроснабжение.
- Охрана.

Для каждой строки фиксировать:

- категорию;
- название объекта;
- единицу измерения;
- значения по `SHELKOVO VILLAGE`, `SHELKOVO FOREST`, `SHELKOVO PARK`, `SHELKOVO RIVER`;
- итог, если его можно надежно посчитать;
- страницу PDF;
- пометку `требует визуальной сверки`, если таблица распознана неоднозначно.

# 060: Выделить спорные расчетные допущения

## Цель

Из методической части полного регламента взять не весь юридический и нормативный текст, а только допущения, которые важны для понимания или проверки тарифа.

## Вход

- Контекст: `docs/reglament/full/000-agent-context.md`.
- Карта страниц: `docs/reglament/full/010-page-map.md`.
- Источник: `docs/reglament/original/full.pdf`, вводная методическая часть.

## Выход

Создать файл:

- `docs/reglament/full/060-calculation-assumptions.md`.

## Что искать

- Единый тариф для четырех поселков.
- Распределение платы по площади участка.
- Нормативный метод и экспертные оценки.
- Общеэксплуатационные расходы `70%`.
- Прибыль `40%`.
- Коэффициент временного проживания `0,5` в расчетах ТКО, если это лучше разобрать вместе с мусорной сметой.

## Что не делать

- Не пересказывать весь список нормативной литературы.
- Не делать юридические выводы.
- Не спорить с документом без фактического основания.

- [x] У каждого спорного допущения есть `почему важно` и `как проверять`.
- [x] Нет изменений в `apps/www`.

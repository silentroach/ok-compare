# 030: Извлечь перечень услуг и периодичность

## Цель

Достать из Приложения №4 перечень услуг, которые ОК заявляет как оказываемые на территории поселка, и их периодичность.

## Вход

- Контекст: `docs/reglament/full/000-agent-context.md`.
- Карта страниц: `docs/reglament/full/010-page-map.md`.
- Источник: `docs/reglament/original/full.pdf`, Приложение №4.

## Выход

Создать файл:

- `docs/reglament/full/030-service-catalog.md`.

## Что извлечь

Группы услуг:

- круглогодично;
- в летний период;
- в зимний период.

Для каждой услуги фиксировать:

- стабильный черновой `id`;
- группу;
- название услуги;
- периодичность;
- страницу PDF;
- короткую цитату или строку источника.

## Acceptance Criteria

- [x] Есть `docs/reglament/full/030-service-catalog.md`.
- [x] Все услуги из Приложения №4 перенесены в структурированную таблицу.
- [x] У каждой услуги есть периодичность и страница PDF.
- [x] Нет попытки сразу сопоставить услуги со сметой; это отдельная задача `050`.
- [x] Нет изменений в `apps/www`.

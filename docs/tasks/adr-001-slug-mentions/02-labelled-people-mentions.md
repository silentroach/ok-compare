# Задача 2: Реализовать упоминания с авторским текстом для людей

## Описание

Расширить `normalizePeopleMentions`, чтобы Markdown-ссылки с адресом вида `@slug` становились ссылками на профиль человека. Реализация должна сохранять текущую обработку `@slug` и `@slug:case`, не трогать обычные ссылки и не обрабатывать `@slug` внутри уже существующего видимого текста ссылки как отдельное каноническое упоминание.

## Критерии приемки

- [x] `[главного по электричеству](@kschemelinin)` превращается в `[главного по электричеству](/people/kschemelinin/)`.
- [x] Видимый текст ссылки остается авторским и не заменяется на имя из профиля.
- [x] Человек из ссылочного упоминания попадает в `mentions` один раз, даже если рядом есть каноническое упоминание того же слага.
- [x] Неизвестный слаг в `[текст](@unknown)` завершает сборку ошибкой с понятным сообщением.
- [x] `[текст](@kschemelinin:gen)` не принимается как валидное упоминание.
- [x] Обычные ссылки, картинки, строчный код и блочный код остаются без изменений.
